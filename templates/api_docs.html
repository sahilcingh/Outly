Outly API Documentation

Use the Outly API to integrate AI-powered B2B recruiter outreach into your own tools, CRM, ATS, or workflows.

Authentication

All API requests require an API key. Generate one at Settings → API Keys.

Pass your key in the Authorization header:

Authorization: Bearer outly_sk_your_key_here

Or use the X-API-Key header:

X-API-Key: outly_sk_your_key_here
⚠️ Keep your API key secret. Do not expose it in frontend code or public repositories.

Base URL

https://outly-me9j.onrender.com/api/v1

Prospect a Company

POST /api/v1/prospect
Research a company and draft a personalized recruiter outreach email.

Request Body (JSON)

FieldTypeRequiredDescription
companystringrequiredCompany name or website URL
industrystringoptionalIndustry hint e.g. "Fintech", "SaaS"
rolestringoptionalCandidate role to offer — auto-detected if omitted

Example Request

curl -X POST https://outly-me9j.onrender.com/api/v1/prospect \
  -H "Authorization: Bearer outly_sk_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "company": "Razorpay",
    "industry": "Fintech",
    "role": "Senior Backend Engineer"
  }'

Example Response

  "company_name": "Razorpay",

Prospect from Resume

POST /api/v1/resume
Find matching companies for a candidate and draft outreach emails for all of them.

Request Body (JSON)

FieldTypeRequiredDescription
resume_textstringrequiredCandidate's resume or LinkedIn summary as plain text
industrystringoptionalIndustry focus e.g. "Fintech", "Healthcare"
max_companiesintegeroptionalNumber of companies to target, 1–10 (default: 5)

Example Request

curl -X POST https://outly-me9j.onrender.com/api/v1/resume \
  -H "Authorization: Bearer outly_sk_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "resume_text": "Senior Java engineer with 6 years experience in Spring Boot, Kafka, PostgreSQL. Previously at Flipkart and Swiggy.",
    "industry": "Fintech",
    "max_companies": 5
  }'

Example Response

{
  "success": true,
  "count": 5,
  "drafts": [
    {
      "company_name": "Razorpay",
      "company_url": "https://razorpay.com",
      "contact_title": "Head of Engineering",
      "role_to_offer": "Senior Java Backend Engineer",
      "subject": "Strong Java Engineer for Razorpay's Backend",
      "body": "Hi...",
      "rationale": "..."
    },
    ...
  ]
}

List Drafts

GET /api/v1/drafts
List your saved drafts. Supports filtering and pagination.

Query Parameters

ParameterTypeDescription
statusstringFilter: draft | approved | sent | rejected
limitintegerMax results (default: 20)

Example

curl https://outly-me9j.onrender.com/api/v1/drafts?status=approved \
  -H "Authorization: Bearer outly_sk_your_key"

Get a Draft

GET /api/v1/drafts/{id}
Retrieve a single draft by its ID.

Example

curl https://outly-me9j.onrender.com/api/v1/drafts/42 \
  -H "Authorization: Bearer outly_sk_your_key"

Error Responses

StatusMeaning
